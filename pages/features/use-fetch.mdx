# Fetch on mount and on key change

You can use `useFetch` to fetch a resource on mount and on key change.

```tsx
XSWR.useFetch(handle) 
```

### Example

```tsx
function useHello() {
  const handle = XSWR.use(getHelloSchema, [])

  XSWR.useFetch(handle)
  return handle
}
```

### Implementation

```tsx
export function useFetch(handle: Handle) {
  const { fetch } = handle

  useEffect(() => {
    fetch()
  }, [fetch])
}
```
