# Fetch on mount and on key change

You can use `useFetch` to fetch a resource on mount and on key change.

```typescript
XSWR.useMount(handle) 
```

### Example

```typescript
function useMyData(init?: MyData) {
  const handle = XSWR.useSingle<MyData>(
    `/api/hello`, 
    fetchAsJson)
  XSWR.useFetch(handle)
  return handle
}
```
