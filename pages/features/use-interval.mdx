# Fetch on a regular interval

You can use `useInterval` to fetch a resource on a regular interval.

```typescript
XSWR.useInterval(handle, 1000) 
```

### Motivation

You know that some data is regularly updated.

### Example

```typescript
function useMyData() {
  const handle = XSWR.useSingle<MyData>(
    `/api/hello`, 
    fetchAsJson)
  XSWR.useFetch(handle)
  XSWR.useInterval(handle, 1000)
  return handle
}
```

### Implementation

```typescript
export function useInterval(handle: Handle, interval: number) {
  const { fetch } = handle

  useEffect(() => {
    if (!interval) return
    const i = setInterval(fetch, interval)
    return () => clearInterval(i)
  }, [fetch, interval])
}
```
